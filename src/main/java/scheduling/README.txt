CSC262 scheduler
Isabella Brody-Calixto
Scheduler simulation

Main is in Main.java, takes one command line argument.
Argument should be a text file.

Input file should be in following format:

"job (name) (arrival time) (duration)"

The following algorithms are simulated:
- Multi-level feedback queue
- Round Robin
- First in first out
- Shortest job to completion
- Shortest job first
- Lottery scheduling
